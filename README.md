# MineSweeper MiniGame

Foundational React project structure for a future minesweeper-style mini game.

## Getting Started

```bash
npm install
npm run dev
```

## Project Layout

- `src/components/Board` – board presentation component placeholder
- `src/components/Card` – individual card placeholder
- `src/components/Controls` – control panel placeholder
- `src/hooks/useGameLogic.js` – hook entry point
- `src/api/simulateBoard.js` – simulated board API placeholder
- `src/utils` – helper stubs
- `src/styles/global.css` – global styles
- `src/App.jsx` – root component wiring the placeholders
- `src/index.js` – application entry point
